## COLUMN(열, 속성, 필드)

- 세로 속성(줄)
- 공통된 값들의 주제

## ROW(행, 레코드, 튜플)

- 하나의 정보

## Primary Key

- 고유한 값(각 행의 구분점으로 사용된다.)
- 중복이 없고 Null을 허용하지 않음
- Null : 아직 어떤 값을 넣을지 모르겠다 라는 뜻을 가진 값

## Foreign Key

- 다른 테이블의 PK를 의미함
- 보통 다른 테이블끼리 관계를 맺을 때 사용함
- 중복이 가능하다
- 위의 예시 표 참고
    - Table A에 없는 값을 Table B의 FK 컬럼에 사용할 수 없다.

## Unique Key

- Null은 허용하지만, 중복은 허용하지 않는 Key
